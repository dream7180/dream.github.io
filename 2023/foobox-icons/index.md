# foobox 图标包 for foobar2000

**【图标包简介】**   
制作精良的 foobox 图标包，关联的格式比 foobar2000 自带的要多一些，从 foobx 7 开始从 foobox 皮肤中分离了出来，不再集成，喜欢的可以单独下载替换 foobar2000 自带的关联图标 （请整仓下载）。
